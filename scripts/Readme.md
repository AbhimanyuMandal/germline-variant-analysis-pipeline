# Scripts

This directory contains the shell and R scripts used in the germline variant analysis workflow.

| Script | Description |
|---------|-------------|
| germline_variant_pipeline.sh | End-to-end germline variant calling pipeline from alignment to annotation |
| merge_maf_files.R | Merge annotated MAF files for downstream analysis |
| average_depth_coverage.sh | Calculate sequencing depth statistics from variant files |
